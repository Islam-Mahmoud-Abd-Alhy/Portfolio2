## 📜 System Overview
A professional, high-performance portfolio designed with a **Cybersecurity/Terminal** aesthetic. This project showcases my skills in Full-Stack development and security research, featuring a fully responsive layout and interactive UI components.

### ✨ Key Features
- 🛡️ **Security-First UI:** Log-file styled articles and "System Locked" sections.
- 📱 **Fully Responsive:** Optimized for all devices from mobile to ultra-wide monitors.
- ⚡ **Framer Motion:** Smooth, hardware-accelerated animations and transitions.
- 📧 **Direct Contact:** Integrated EmailJS for real-time communication.
- 🌊 **Modern Styling:** Glassmorphism effects using Tailwind CSS.
